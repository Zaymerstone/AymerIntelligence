## 🌟 Overview

In today's fast-paced market, manual research is a bottleneck. Aymer Intelligence solves this by deploying a "Strategic Swarm" of autonomous agents that scout, analyze, and visualize data simultaneously. 



Instead of hours of manual labor with LLM, which are not meant for that, users get:

- **Instant Market Context**

- **Automated Financial Projections**

- **Risk Assessment via Weighted Algorithms**



---



## 🛠️ Key Features

- **Autonomous Scouting:** Real-time web intelligence gathering across global sources.

- **Dynamic Data Visualization:** Bloomberg-style dashboards with interactive charts.

- **Agentic Orchestration:** Specialized agents (Researcher, Analyst, Reporter, Extractor) working in a coordinated pipeline.

- **Executive Reporting:** Professional synthesis of complex data points into actionable insights.



---



## 🏗 System Architecture

The platform operates on a high-performance bridge between AI reasoning and modern frontend engineering:



1. **The Brain (Airia AI):** Orchestrates multiple LLM-based agents with custom prompt guardrails.

2. **The Logic (TypeScript/React):** Parses agentic JSON outputs into structured data models.

3. **The Interface (Tailwind/Lucide):** Provides a distraction-free, "Intelligence-First" UI/UX.

## 🤖 The Intelligence Swarm

Aymer Intelligence utilizes a nested agent architecture:



* **🕵️ Market Scout:** Performs deep-web research and competitor analysis.

* **📊 Financial Analyst:** Processes ROI, market caps, and fiscal trends.

* **📝 Chief Reporter:** Synthesizes raw findings into a narrative structure.

* **📐 Data Extractor:** Ensures 100% JSON integrity for UI components.

## 🚀 Getting Started



### Prerequisites

- [Bun](https://bun.sh) (recommended) or Node.js

- Airia API Key



### Installation

1. Clone the repository:

   ```bash

   git clone [https://github.com/Zaymerstone/aymer-intelligence.git](https://github.com/Zaymerstone/aymer-intelligence.git)

   cd aymer-intelligence/app

2. Install dependencies:

    ```bash
    bun install
    ```
  
3. Set up environment variables:

    Create a ```.env``` file in the ```app``` folder and add:
    ```VITE_AIRIA_API_KEY=your_key_here```

4. Run development server

    ```bash
    bun dev
    ```
